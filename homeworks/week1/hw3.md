## 教你朋友 CLI

**Command Line 是什麼?** 另一種操縱電腦的方法
* Graphical User Interface (GUI)：圖形化介面 
* Command Line Interface (CLI)：命令列介面

環境設置：安裝 Git ([官網](https://git-scm.com/))，使用 Git-bash
1. 建立資料夾：`mkdir wifi`
2. 進入資料夾：`cd wifi`
3. 建立檔案：`touch afu.js`
4. 查看是否有新增成功：`ls`
